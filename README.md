Fairy Tail Inspired Weather Application


A cozy weather website based off of my favorite anime, Fairy Tail. Rather than relying on traditonla weather app UI, current weather is displayed using 3 frame pixel art animations 

Weather Events
- Rain 
- Snow
- Day
- Night
- Each conditon has its own animation!
  
Tech Stack
- HTML
- CSS
- JS
- Weather API

Display/Art
- Website Used for Art Creation: Pixilart
- Used PNGs which allows the site to give off an animation. GIFs not used due to them creating issues with the color palettes. PNGs preserves intended colors.

Potential future implementations include more weather events and animations.

What I learned
- How to draw using pixel art. (First time using pixel art)
- How to use APIs
- Creating animations using PNGs
- Translating weather into visualizations.
